# Security Policy

LastLight is experimental software for offline retrieval from local Markdown files. It does not use network access, cloud APIs, telemetry, a browser, a web server, or background services.

## Reporting Issues

Report security concerns privately to:

edujbarrios@outlook.com

Please include:

- affected version or commit
- operating system
- steps to reproduce
- expected and observed behavior

## Scope

Security-relevant issues include unsafe file handling, unexpected network behavior, misleading confidence behavior, and bugs that could cause LastLight to present unsupported information as reliable.
